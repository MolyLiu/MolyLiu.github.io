Redis基本数据类型(四)Hashes常用指令

本文总共645个字 | 您是第位看到它们的小伙伴

Hashes

redis中hashes可以看做是具有string key和string value的map容器。很适合存储==值对象==的信息

常用指令

hset key field value

为指定的key设定field/value对，如果key不存在，则创建再执行，如果field已存在，则覆盖

hget key filed

返回指定key中指定filed的关联值

hexists key field

判断指定key中指定field是否存在。

hlen key

获取该key所包含的field的数量。

hdel key field ..field

删除一个字段。返回删除成功的数量

hsetnx key field value

只有当参数中的key/field不存在的时候进行操作，否则不会进行任何操作

hincrby key field increment

原子性加 increment值，返回运算之后的值

hgetall key

获得所有的field/value对，格式为一个field 一个value

hkeys key

返回key包含的所有field

hvals key

返回key所包含的所有value名

hmget key field..filed

返回指定的fields值

hmset key field value ..field value

批量添加fidle/value 对，如果field已经存在 ,则用新值覆盖。
