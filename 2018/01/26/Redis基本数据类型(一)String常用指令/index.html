Redis基本数据类型(一)String常用指令

本文总共585个字 | 您是第位看到它们的小伙伴

String

常用指令

set key value

设置key为指定value值 若key已存在则覆盖其原来的值

get key

返回key值

append key value

在key的原来value后增加新的value 如果key不存在 则创建并赋值 返回value的长度

decr key

key对应的value原子性减1 返回现在的value值 如果key不存在 则创建key value为0 后在执行 若value值不能转换为整型返回错误原因

incr key

同上条 原子性加1

decrby key value

同上条 原子性减value

incrby key value

同上条 原子性加value

getset key value

返回key原值 并将key赋新值

strlen key

返回key的value的长度 如果value不是string类型则返回错误

setnx key value

若key不存在则等同于set 若key存在则不进行操作 返回0/1

setrange key offset value

对key的value从第offset位开始进行替换 替换成新value

getrange key start end

截取start end之间部分，

在指令前加m 后可以进行批量处理
